2. Create examples/output/README.md explaining output format:

**Note at top:** Real outputs require running the pipeline with GPU. This README documents what to expect.

**Output Structure:**
```
{job_id}/
  output/
    reconviagen/           # If ReconViaGen model selected
      mesh.obj             # 3D mesh in OBJ format
      mesh.mtl             # Material file
      mesh.glb             # GLB format (primary)
      texture_kd.png       # Diffuse texture
    nvdiffrec/             # If nvdiffrec model selected
      mesh.obj
      mesh.mtl
      mesh.glb
      texture_kd.png
    preview/
      front.png            # Preview renders
      back.png
      left.png
      right.png
      top.png
      bottom.png
      wireframe_front.png  # Wireframe overlays
      ...
    quality.json           # Quality metrics
```

**quality.json format:**
```json
{
  "status": "normal|warning|failure",
  "metrics": {
    "psnr": 28.5,
    "ssim": 0.89
  },
  "thresholds": {
    "psnr": {"normal": 25, "warning": 20},
    "ssim": {"normal": 0.85, "warning": 0.75}
  }
}
```

**Status Classification:**
- normal: PSNR >= 25dB AND SSIM >= 0.85
- warning: PSNR >= 20dB AND SSIM >= 0.75 (but below normal)
- failure: Below warning thresholds

**Download Format:**
When downloading via GET /jobs/{job_id}/download, you receive a ZIP containing all output files.
